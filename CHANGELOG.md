# Changelog

All notable changes to this project will be documented in this file.

## Current version

### Added

- Complete full pipeline runner.
- Historical solar and battery optimization.
- Forecasting pipeline using Machine Learning.
- Forecast-based solar and battery optimization.
- Historical vs forecast-based optimization comparison.
- Final results summary report.
- Configuration summary report.
- Outputs index report.
- PVGIS solar generation integration.
- UCI household consumption dataset preprocessing.
- Configurable electricity tariff profiles.
- Battery simulation.
- Economic grid search optimization.
- Generated reports and plots.
- Makefile with common project commands.
- Pytest test suite.
- Ruff lint checks.
- GitHub Actions workflow for lint and tests.
- Folder-level documentation.
- Setup guide.
- Contributing guide.
- MIT license.

### Changed

- Improved README documentation.
- Added project highlights and quick summary.
- Added documentation for learning purpose and AI assistance.
- Improved project reproducibility.
- Improved `.gitignore` for development caches and generated data.

### Notes

The project is currently focused on a complete reproducible portfolio pipeline using public household consumption data and PVGIS solar generation data.

Future work may include hourly electricity price CSV files, PVPC/ESIOS integration, weather-based forecasting, more forecasting models and a Streamlit dashboard.

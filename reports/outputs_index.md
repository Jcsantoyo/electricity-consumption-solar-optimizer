# Project outputs

Solar data source: PVGIS solar data (data/raw/pvgis_hourly_linares_1kw_2020.csv)

Electricity price mode: Time-of-use electricity prices from tariff profile 'flat_price'

This folder contains the generated reports from the electricity consumption solar optimizer.

## CSV reports

### `grid_search_results.csv`

Full historical grid search results. Each row represents one combination of solar peak power and battery capacity, including investment cost, annual savings, payback period, grid import, annual grid import, solar surplus and self-sufficiency.

### `best_scenarios.csv`

Summary table with the two selected historical optimal scenarios: one by minimum payback period and one by maximum self-sufficiency.

### `best_scenario_timeseries.csv`

Hourly energy flow table for the best economic historical scenario. It includes consumption, solar generation, self-consumption, battery charge, battery discharge, grid import, solar surplus and battery state.

### `forecast_results.csv`

Forecasting results for the test period. It includes the actual consumption and the predicted consumption generated by the Machine Learning model.

### `forecast_feature_importance.csv`

Feature importance values from the Random Forest forecasting model. It shows which variables are most relevant for predicting household electricity consumption.

### `forecast_model_comparison.csv`

Comparison between forecasting models using error metrics such as MAE and RMSE.

### `forecasted_consumption_for_optimization.csv`

Forecasted consumption converted into the standard project input format so it can be used by the solar and battery optimization pipeline.

### `forecast_optimization_results.csv`

Full grid search results using forecasted consumption instead of historical consumption.

### `forecast_optimization_best_scenarios.csv`

Best economic and best self-sufficiency scenarios obtained from forecast-based optimization.

### `historical_vs_forecast_optimization.csv`

Comparison table between the best historical optimization scenarios and the best forecast-based optimization scenarios.

## Text reports

### `configuration_summary.md`

Summary of the configuration used by the project pipeline, including input data paths, PVGIS settings, grid search parameters, battery model assumptions, economic assumptions and the active tariff profile.

### `final_results_summary.md`

Readable final summary of the main results produced by the full pipeline. It compares the best historical and forecast-based optimization scenarios.

### `summary.txt`

Readable summary of the best historical economic scenario and the best historical self-sufficiency scenario.

### `outputs_index.md`

Index describing the generated reports and plots.

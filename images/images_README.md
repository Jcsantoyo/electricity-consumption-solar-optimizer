# Images documentation

This folder contains generated plots produced by the project pipeline.

Most files in this folder are outputs. They can be regenerated by running:

```bash
make pipeline
```

or:

```bash
python scripts/run_full_pipeline.py
```

---

## Historical optimization plots

### `main_payback_grid_search.png`

Shows how the payback period changes depending on:

- Solar peak power
- Battery capacity

This plot helps identify the most economically attractive solar and battery configuration.

### `main_self_sufficiency_grid_search.png`

Shows how energy self-sufficiency changes depending on:

- Solar peak power
- Battery capacity

This plot helps identify which configuration reduces dependence on the electricity grid the most.

### `best_scenarios_comparison.png`

Compares the best economic scenario with the best self-sufficiency scenario.

It highlights that the most profitable configuration is not necessarily the one with the highest self-sufficiency.

### `best_scenario_timeseries.png`

Shows hourly energy flows for the selected best economic scenario.

It includes:

- Consumption
- Solar generation
- Grid import

### `best_scenario_battery_state.png`

Shows the battery state of charge over time.

This is useful for understanding how the simulated battery charges and discharges.

### `best_scenario_cumulative_energy.png`

Shows cumulative energy flows over time.

It includes accumulated values such as:

- Consumption
- Solar generation
- Grid import
- Solar surplus

## Forecasting plots

### `consumption_forecast_actual_vs_predicted.png`

Compares actual electricity consumption with predicted consumption from the Machine Learning model.

### `forecast_feature_importance.png`

Shows which features are most important for the Random Forest forecasting model.

### `forecast_model_comparison.png`

Compares forecasting models using error metrics such as MAE.
